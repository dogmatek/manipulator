программма управления манипулятором.

Сервеная часть позволяет редактировать программу-скрипт по упарвлению манипулятором.
Команды:

(GET) hostname:8081/servo - отображает все команды по управлению манипулятором
(GET) hostname:8081/servo/{id} - отображает коману по управлению манипулятором c указанным id по управлению манипулятором
(GET) hostname:8081/servo/program/{id} - отображает все команды по управлению манипулятором программы ID
(POST) hostname:8081/servo/ - добавляет  коману по управлению манипулятором.
{
        "pid": 1,       - ID программы
        "ordering": 4,  - номер позиции команды
        "state": 1,     - Состояние 1- опубликовано
        "period": 1000, - Задержка перед выполнением миллисекунды
        "pin00": 30,    - поворот сервпривода значение от 0 до 100
        "pin01": 60,    - значение 0 - оставить в предыдущем положении
        "pin02": 50,    - вего 6 сервоприводов от 00 до 05
        "pin03": 20,
        "pin04": 0
        "pin05": 0
    }
(DELETE) hostname:8081/servo/{id} - удаляет команду по управлению манипулятором.


(GET) hostname:8081/program - отображает все программы по упарвлению манипулятором
(POST) hostname:8081/program - добавляет программу
(DELETE) hostname:8081/program/{id} - удаляет программу по управлению манипулятором.

(PUT) hostname:8081/servo/run/{id} - Запускает программу с ID
(PUT) hostname:8081/servo/rotation/{pin}/{rotation} - поворачивает сервопривод по указанному pin в позицию rotation (1-100)


(POST) hostname:8081/login  -  Авторизация пользователя (user/user admin/admin)
поля для авторизации username и password
